# React Quiz Builder

A dynamic quiz builder built with React.  
Users can create quizzes with multiple questions and options, select correct answers, and switch to solve mode to attempt the quiz. Data persists in the browser using localStorage.

## Features
- Builder mode + Solve mode
- Component-based structure (`QuestionCard`, `OptionInput`)
- LocalStorage persistence
- Dynamic question & option management
- Score calculation

## Tech Stack
- React (functional components)
- JavaScript
- Vite
- Browser localStorage

## How to Run Locally
1. Clone the repo
2. Install dependencies: `npm install`
3. Start development server: `npm run dev`
4. Open browser: `http://localhost:5173`
